HOP HOUSE — Быстрый запуск (без команд)

1) Установите Node.js (если не установлен): https://nodejs.org/en (LTS версия)
2) Windows: дважды кликните start_windows.bat
   macOS/Linux: запустите start_mac_linux.sh (в терминале: bash start_mac_linux.sh)
3) Откроется страница меню: http://localhost:3000
4) Добавляйте блюда кликом, оформляйте заказ, указывайте столик (1–100), подтверждайте.
5) Телеграм-оповещения (по желанию): создайте файл .env рядом с server.js со строчками:
   TELEGRAM_BOT_TOKEN=ВАШ_ТОКЕН
   ADMIN_CHAT_ID=701300453
   PORT=3000
   После этого перезапустите скрипт запуска.

Если телеграм не настроен, заказы всё равно подтверждаются, а текст заказа печатается в окне сервера (демо-режим).
